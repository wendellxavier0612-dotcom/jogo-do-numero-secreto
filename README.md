# Jogo do número secreto

Jogo de adivinhação desenvolvido em JavaScript, onde o objetivo é descobrir um número secreto sorteado aleatoriamente.

## 🎮 Como funciona

- Um número aleatório é sorteado dentro de um intervalo definido
- O jogador tenta adivinhar o número
- A cada tentativa, o jogo indica se o número secreto é maior ou menor que o chute
- É possível reiniciar o jogo a qualquer momento

## 🛠️ Tecnologias

- JavaScript
- HTML
- CSS
